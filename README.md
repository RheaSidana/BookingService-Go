# BookingService-Go

<h1>1. Clone the repo </h1>
```
git clone https://github.com/RheaSidana/BookingService-Go.git
```

<h1>2. Run the command:</h1>
```
go mod tidy
```

<h1>3. DB operations:</h1>

```

```

```
brew services start/stop postgresql
```
